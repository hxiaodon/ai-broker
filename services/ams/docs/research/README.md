## 关键数据点总结

### 国际标准
- **FATF Recommendation 12**：PEP 定义包括"重要公共职务"的政府官员及其直系亲属
- **SFC 2023 修订**：将中国大陆官员明确列为 Non-HK PEP，**省级及以上强制 EDD**

### 行业实践
- **Interactive Brokers**：省级自动 EDD，市级按关键词判断（拒绝率 3-5%）
- **Saxo Bank**：分层标准（Tier 1-4），全球一致（拒绝率 2%）
- **国内持牌券商**：至少省级自动分类为 PEP（共识）

### 监管处罚风险
- **PEP 遗漏处罚**：$500K-$2.5M/案件（SFC 2019-2024 记录）
- **案例**：某港资券商因遗漏"省副级"官员配偶，被罚 $2.5M
- **处罚趋势**：SFC 对 PEP 遗漏零容忍；重视流程文件化

### 成本效益
| 项目 | 金额 | ROI |
|------|------|-----|
| **人工成本（年）** | $600K | vs $500K-$1M+ 处罚 |
| **工具成本（年）** | $300K | vs $500K-$1M+ 处罚 |
| **开发成本（一次性）** | $200K | vs $500K-$1M+ 处罚 |
