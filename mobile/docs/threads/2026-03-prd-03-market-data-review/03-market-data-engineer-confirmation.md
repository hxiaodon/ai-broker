## 工程实现准备

基于 PRD v2.3，Market Data Service 可开始以下开发工作：

1. **ETF 大盘指数数据接入**
   - SPY/QQQ/DIA 作为普通股票 symbol 接入
   - 复用现有 quote 推送机制

2. **换手率计算模块**
   - 集成 Polygon Fundamental API
   - 每日收盘后更新 shares_outstanding 缓存
   - 实时计算换手率 = volume / shares_outstanding × 100%

3. **中文搜索索引构建**
   - 构建 Top 1000 美股中文名/拼音映射表
   - 支持代码前缀、英文名、中文名、拼音首字母搜索
   - Debounce 300ms + 最少字符限制

4. **WebSocket 双轨推送**
   - 注册用户: tick 级实时推送 (<500ms)
   - 访客: T-15min 快照，每 5 秒推送
   - 延迟标识: 每处价格旁显示"延迟15分钟"
