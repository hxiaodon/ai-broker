v3-final 设计规范

  • • 涨跌色: 可配置 (红涨绿跌 / 绿涨红跌)
  • • 圆角: 8px(小) / 12px(卡片) / 16px(弹窗)
  • • 最小点击区域: 44x44px
  • • 视口宽度: 375px (iPhone 标准)
  • • Text Secondary: #595959 (WCAG AAA)
  • • 动画: 进入ease-out(0.3s), 退出ease-in(0.2s)
  • • 加载: 骨架屏替代Spinner
