## 1. 产品定位

**目标用户**: 美港股投资者（个人投资者为主）
**核心价值**: 便捷交易 + 实时行情 + 资产管理

**竞品参考**
- Robinhood: 极简设计，2 步交易流程
- Tiger Brokers: 专业行情 + 社区
- Futu: 功能全面，FPS 实时入金
- Webull: 图表强大，50+ 技术指标

## 4. 页面设计

### 4.1 登录/注册/KYC 完整流程

**完整流程图**
```
启动页 → 登录页 → 手机验证 → 设置密码
                ↓
              注册页 → 身份信息 → 证件上传 → 人脸识别
                    → 地址证明 → 投资者评估 → 风险披露
                    → 协议签署 → 完成
```

**🆕 KYC 认证详细步骤**

1. **身份信息**
   - 姓名、身份证号/护照号
   - 出生日期、国籍
   - 同名验证（与银行账户匹配）

2. **证件上传**
   - 身份证正反面 / 护照首页
   - OCR 自动识别
   - 活体检测防伪

3. **人脸识别**
   - 活体检测（眨眼、转头）
   - 与证件照比对
   - 防止照片/视频攻击

4. **地址证明**
   - 水电费账单 / 银行对账单
   - 3 个月内有效
   - 地址与身份证一致

5. **投资者适当性评估**
   - 投资经验（年限、品种）
   - 风险承受能力（保守/稳健/激进）
   - 财务状况（年收入、净资产）
   - 投资目标（保值/增值/投机）

6. **风险披露**
   - 市场风险披露
   - 订单类型风险说明
   - PDT 规则详细说明
   - Best Execution 披露
   - PFOF 披露（如适用）

7. **协议签署**
   - 客户协议
   - 风险披露声明
   - 隐私政策
   - 电子签名

**登录页设计要点**
- 手机号 + 验证码登录（主要方式）
- 邮箱 + 密码登录（备用方式）
- 生物识别（Face ID / Touch ID）快速登录
- "忘记密码" 和 "注册账号" 入口
- 🆕 "先浏览后注册"选项（降低门槛）

### 4.4 交易下单页（简化流程）

**🆕 简化后的买入流程（3 步）**

**步骤 1: 输入订单（合并输入和预览）**
```
┌─────────────────────────────────┐
│  ← 买入 AAPL                    │
├─────────────────────────────────┤
│  当前价格: $175.23              │
│  可用资金: $10,000.00           │
├─────────────────────────────────┤
│  [市价单] [限价单] [止损单]    │ ← Tab 切换
├─────────────────────────────────┤
│  限价: $ 175.00                 │ ← 价格输入
│  数量: 100 股                   │ ← 数量输入
│  [10] [50] [100] [全部]        │ ← 快捷按钮
├─────────────────────────────────┤
│  预估成本: $17,500.00           │
│  手续费:   $5.00                │ ← 🆕 实时计算
│  合计:     $17,505.00           │
├─────────────────────────────────┤
│  ⚠️ 风险提示                    │ ← 🆕 风控提示
│  • 限价单可能无法成交           │
│  • 市场波动风险                 │
├─────────────────────────────────┤
│  [滑动确认买入 →]               │ ← 🆕 滑动确认
└─────────────────────────────────┘
```

**步骤 2: 生物识别验证**
- Face ID / Touch ID 验证
- 替代密码输入，更快捷

**步骤 3: 提交成功**
- Toast 提示 + 震动反馈
- 自动跳转到订单列表

**🆕 交易防护机制**
- 2 秒内禁止重复提交（防抖动）
- 可用资金不足：禁用按钮 + 红色提示
- PDT 规则：强制阻断（不只是提示）
- 大额订单（>$10,000）：二次确认弹窗
- 异常波动（价格偏离 5%）：警告提示
- 市场休市：显示下次开盘时间

**🆕 订单确认弹窗内容**
- 订单摘要（股票、数量、价格）
- 预估成交价（市价单显示滑点范围）
- 手续费明细
- 风险提示
- Best Execution 披露

### 4.7 出入金页面（增强合规）

**🆕 银行卡验证流程**
1. 输入账户名、账号、银行名
2. 同名验证（与 KYC 姓名匹配）

**🆕 即时入金方案**
- 使用 Plaid 即时验证

- 限额 $1,000（首次）/ $5,000（老用户）

## 5. 设计规范

### 5.1 颜色系统

**🆕 中性色（调整对比度）**
- Text Secondary: `#595959` (原 `#8C8C8C`，提升至 7:1 对比度)

**🆕 暗色模式**
- Primary: `#177DDC` (降低蓝光)
- Background: `#141414`
- Text Primary: `#FFFFFF`

### 5.2 字体系统

**🆕 大字模式**
- 所有字号 +2px/sp
- 关键按钮增大至 48x48px

### 5.3 圆角

- 小圆角: 8px (统一)
- 中圆角: 12px (卡片)
- 大圆角: 16px (弹窗)
