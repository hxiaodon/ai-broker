## 一、产品定位

面向中国大陆及香港零售投资者的跨境证券经纪移动应用，Phase 1 聚焦美股交易全链路闭环，Phase 2 延伸至港股。

| 维度 | 内容 |
|------|------|
| 目标用户 | 有海外投资需求的中国大陆（+86）/ 香港（+852）居民 |
| 核心价值 | 合规开户 · 实时行情 · 便捷交易 · 快速出入金 |
| 监管框架 | Phase 1：SEC/FINRA（美国）；Phase 2 追加 SFC/AMLO（香港） |
| Phase 1 市场 | 美股（NYSE/NASDAQ）；港股展示入口 "敬请期待" |

## 三、Phase 1 范围边界

### 包含（In Scope）

- 手机号 OTP 登录（+86 / +852）
- 7 步 KYC（不含活体检测，人工审核替代）
- 美股实时行情（注册用户） + 15 分钟延迟行情（访客）
- 市价单 / 限价单，DAY / GTC
- 美股常规交易时段 + 盘前/盘后（仅限价单）
- 入金：USD ACH / Wire；出金：USD ACH / Wire
- 银行卡管理（最多 5 张，同名验证）
- 持仓、P&L、订单历史
- Admin Panel：KYC 审核、出金审批、用户管理、订单监控、系统看板

### 不包含（Out of Scope → Phase 2+）

- 港股交易（行情入口保留，点击显示"敬请期待"）
- 活体检测 SDK（Onfido/Sumsub/Jumio）
- 融资融券（Margin Account）
- 即时入金（平台垫资）
- 止损/止损限价/追踪止损订单
- 委托修改（Phase 1 仅支持撤单）
- FX 换汇
- 交易确认书 / 月度对账单（文档显示"敬请期待"）
- 多语言（Phase 1 仅中文）

## 四、平台与技术约束

| 项目 | 约束 |
|------|------|
| iOS | 最低 iOS 16（Swift Charts 原生支持，覆盖率 95%+） |
| Android | 最低 API 26（Android 8.0，覆盖率 95%+） |
| 后端 | Go 1.22+，PostgreSQL 16+，Redis 7+，Kafka |
| 财务计算 | 全链路禁止浮点数（Go: shopspring/decimal，Swift: Foundation.Decimal，Kotlin: BigDecimal） |
| 时间 | 所有存储/传输使用 UTC，仅显示层转换用户时区 |
| 安全 | JWT（RS256，15 分钟有效期）+ HttpOnly Cookie（7 天 Refresh Token） |

## 五、KYC 分级与资金限额

| 等级 | 条件 | 单笔限额 | 日限额 | 月限额 |
|------|------|----------|--------|--------|
| Tier 1 | KYC 提交中 / 补件待处理 | $5,000 | $10,000 | $50,000 |
| Tier 2 | KYC 全部通过 + W-8BEN 签署 | $50,000 | $100,000 | $500,000 |
| Tier 3（Phase 2） | 机构 / 高净值 | 另定 | 另定 | 另定 |

## 六、关键外部接入

| 类别 | Phase 1 方案 |
|------|-------------|
| 美股行情 | Polygon.io（注册用户实时，访客延迟 15 分钟） |
| 港股行情 | HKEX OMD（Phase 2） |
| 交易执行 | FIX Protocol，通过经纪商/做市商 UAT 认证 |
| 银行验证（USD） | 微存款验证 / Plaid（Phase 2 快捷验证） |
| KYC 活体 | Phase 2：Onfido / Sumsub / Jumio（评估中） |
| SMS 通道 | Phase 1 供应商待确认（本周内） |
